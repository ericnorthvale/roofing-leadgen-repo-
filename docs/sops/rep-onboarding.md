# SOP: Sales Rep (§3508) & Canvasser Onboarding

**Why:** the growth engine is the sales team. Onboarding must be repeatable so
adding a rep is a checklist, not a tribal-knowledge transfer.

## §3508 commission rep (1099)

- [ ] Signed **§3508 Direct Seller Agreement** + confidentiality/IP assignment
      (AI-drafted, attorney-reviewed) **before any selling activity**
- [ ] W-9; commission structure acknowledged [NEEDS INPUT: commission schedule]
- [ ] HighLevel seat + pipeline access; trained on crm-hygiene + sales-process SOPs
- [ ] Brand-voice + compliance briefing (no door-knock §27.02 violations, no
      deductible-waiver talk, no fabricated claims)
- [ ] Territory / lead-routing rules [NEEDS INPUT: round-robin vs. territory]

## W-2 canvasser

- [ ] Standard W-2 hire paperwork (I-9, W-4, direct deposit, handbook ack)
- [ ] Pay + appointment-bonus structure [NEEDS INPUT: $/hr + bonus]
- [ ] Script + compliance briefing; canvassing only in approved neighborhoods

## Standard

- No selling/canvassing before the signed agreement + compliance briefing are done.
- Every rep follows the same pipeline + Lead Source discipline (crm-hygiene SOP).
