# SOP: Review Generation (Real Reviews Only)

**Why:** reviews drive local SEO + trust. They must be **genuine** — fabricating
or incentivizing-for-positive reviews violates the FTC 2024 rule and Google policy.

## Process

1. **Trigger:** job marked complete (final walk-through done, packet delivered).
2. **Ask the happy customer in person** at close — the single biggest lever.
3. **Send the request** (SMS + email) with a direct Google review link.
   [NEEDS INPUT: Google review short-link once GBP is live.]
   Automate via HighLevel once connected (Phase 2 review engine).
4. **Respond to every review** (positive and negative) professionally.
5. Approved reviews surface on the site via the reviews collection / Places API
   (real only; `aggregateRating` schema only when genuine reviews exist).

## Hard lines

- Never write, buy, or incentivize reviews. Never gate the ask to only-happy
  customers in a way that violates FTC rules — ask all completed customers.
- Never post a review the customer didn't actually write.
