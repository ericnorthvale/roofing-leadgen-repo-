# SOP: Subcontractor Crew Onboarding

**Why:** consistent crew quality + clean compliance paperwork protect the brand,
the warranty, and the company from liability.

## Onboarding checklist (before any work)

- [ ] Site visit + reference check + review past work
- [ ] Signed **Master Subcontractor Agreement** (the AI-drafted, attorney-reviewed template)
- [ ] Current **General Liability** COI on file (Northvale as certificate holder)
- [ ] Current **Workers' Comp** COI **or** a filed **DWC-083** waiver
- [ ] W-9 collected; independent-contractor classification confirmed
- [ ] Per-square pricing agreed in writing [NEEDS INPUT: pricing schedule]
- [ ] Quality + cleanup standard reviewed (magnet sweep, photo-doc via CompanyCam)

## Ongoing

- COIs and DWC-083s tracked with renewal dates; no expired coverage works a job.
  [NEEDS INPUT: where the COI/DWC-083 tracker lives — JobNimbus or a sheet.]
- Agreements re-signed annually.

## Hard lines

- No crew works without current GL + WC/DWC-083 on file. No exceptions.
