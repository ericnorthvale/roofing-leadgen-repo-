# SOP: Insurance Claim & Supplement Workflow

**Why:** supplements are where storm-roofing margin is won or lost. Consistency
here = revenue. We are the **contractor, not a public adjuster** (Tex. Ins. Code
§4102) and we **never** waive deductibles (insurance fraud).

## Steps

1. **Pre-claim inspection.** Honestly assess whether damage justifies a claim — a
   denied claim is worse than none. Document with the forty-photo packet + hail/wind
   indicators (A/C coil dings, gutter-apron dents, soft-metal, granule loss, ridge lift).
2. **Homeowner files the claim.** We support; we do not negotiate as a PA.
3. **Adjuster meeting.** Attend on-site with documentation + a written scope of loss
   aligned to Xactimate line items.
4. **Supplement.** If the adjuster's scope misses code-required items (decking,
   ice-and-water, drip edge, etc.), document and submit the supplement to the carrier.
   [NEEDS INPUT: standard supplement line-item checklist by roof type.]
5. **Approval → production.** On agreed scope, schedule install (JobNimbus once live).
6. **Close packet.** Final forty-photo packet for the homeowner's file.

## Hard lines (compliance)

- No deductible waivers/discounts of any kind.
- No acting as a public adjuster or negotiating the claim for the homeowner.
- All door-to-door/contract steps honor the Texas 3-day rescission (§27.02).
